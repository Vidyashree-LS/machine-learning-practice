Logistic Regression 
what is Logistic Regression?
* It is supervised learning algorithm which comes under regrression and used for classfication scenario

what is calssification algorithm?
* It is a supervised learning technique which predicts categorical values

ex:
* Spam detection--->here classification algirithm predicts spam and not spam labels
* sentoment analysis---> here classfication algorithm predicts positive negative and neutral

How to select algorithms ?
* If target column values are continuos then regression algorithms 
* If target column values are categorical then we use classification algorithms

what logistic means?
* Logistic ---> Sigmoid function
* this sigmoid function mostly we use for binary class labels(target column values)
* Here it finds s curve pattern
* Internally it uses P = 1/(1+e**-z)
* It returns not exactly 0 and 1 but it gives between 0 and 1 and it uses the default threshold as 0.5
* >=0.5 ==> 1 and <0.5 ==> 0

Why it comes under regression?
* Internally it uses regression formula (z)
* z--->w1x1 + w2x2 + c
* here in logistic regression, weights are not slope 
* weights are importance given to the feature

Why not linear regression for binary values?
* because it predicts continuos vlaues and it not possible to fix the threshold

How weights calculated in logistic regression ?
(it calculates weights with the help of optimization technique called gradient descent)
* First it ramdomly initializes the weights between 0 and 0.5 
* and later with the help of gradient descent it finds the best weights 
* To find or optimize the best weights internally it takes mulitiple iterations

What and all techniques logistic regression uses internally?
1. Loss function
2. Cost function
3. Sigmoid function
4. Regularization techniques (l1,l2, elastic net)
5. Gradient descent

* Internally it uses softmax instead of sigmoid for more than two labels
* sag and saga we use when we have more features ...that scenario we get during sentence based classification

what is Regularization?
* It is a ML technique which hepls us to reduce the variance with the help of penalties(L1,L2, elastic net)

What is generalization?
* How well the model captures the pattern of unseen data 
* if the model captures the pattern of unseen data then the model is generalized well
* If model fails to caaptures unseen data pattern then the model is not generalized well(overfit ,solution: Regilarization)

Regularization has 3 penalties l1,l1,elastic net

what is lasso(l1)?
* It is a Regularization penalty which selects important features by pushing the weights towards to zero and exactly to zero
* If the weight is zero which means that feature is not important 
* ex: y = m1x1+m2x2+c
      m1 = 0, m2 = 5, c=10
      x1 = 6, x2 = 6
* with the help of gradient descent to pushes the weights towards zer0

when to use lasso?
* when we have more features
* For sentence based classification
* when we dont know the important features 

what is l2 -> Ridge?
* it is Regularization penalty which keeps all the columns by shrinking the weights 
* It shrinks the weights but never forces to zero

When to ridge?
* when all the feature columns are important
* when multicollineariy problem occurs 

what is elastic net -> l1 +l2?
* it is a Regularization penalty which uses both l1 and l2 
* it selects important features and also handles multicollineariy problem
* it uses one parameter called l1_ratio 
   ex:  l1_ratio = 0.3 and remaining for l2
        l1_ratio = 0 == l2
        l1_ratio = 1 == l1

* Regularization techniques uses gradient descent to find the best weights and rduces the variance

Precision_score
* It is classification metrics which tells out of all the predicted values how much percentage values are cerrectly predicted 

precision score for positive label:
* out of all the predicted positive labels how much percentage labels are predicted as positive
* formula: TP/(TP+FP)

* forst conclusion ---> out of all the predicted positive labels (400) , 50% samples are correctly predicted as positive
+second conclusion for sapma and not spam ----> out of all the predicted spam labels(320), 93.75% samples are correctly predicted as spam

precision score for negative labels:
* out of sll the predicted negative labels how much percentage labels are predicted as negative
* formula: TN/(TN+FN)

Syntax:
from sklearn.metrics import Precision_score
Precision_score(actual_output, predicted_output,pos_label=1)

for train data:
Precision_score(ytrain,y_pred_train)

for test data:
Precision_score(ytset, y_pred_test)

Recall 
* it is a classification metrics which tells out of all the actual values how many values are correctly predicted

Recall score for negative labels:
* out of all the actual negative labels how maany labels are correctly predicted as negative
Formual :
=TN/(TN+FP)

Recall score for positive labels:
* out of all the actual positive labels how many labels are correctly predicted as positive
Formula:
=TP/(TP+FN)

example:
negative: 100/300 = 33.33%
positive: 200/700 = 28.5%

Accuracy score
what is accuracy score?
* It is clasiification metrics which tells out of all the samples how many samples are correctly predicted.

when not to use accuracy score?
* If the class labels are imbalanced we should not use accuracy score
* because it gives wrong conclusions
* for example :        0        1
              (1000)0  1000     0  ==> 100%
              (200) 1  200      0  ==> 0%
       In the above example it is not predicting not even 1 label as positive 

when to use accuracy score?
* when we have balanced class labels
